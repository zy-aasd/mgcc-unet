# mgcc-unet
Multi-granularity Cognitive Computing-based UNet for Medical Image Segmentation

Pretrained weights can be downloaded here. [Baidu](https://pan.baidu.com/s/1YLorPPLHULCl2mwXzEQ1EA?pwd=0612)
The Synapse and ACDC datasets can be downloaded from TransUNet [TransUNet](https://github.com/Beckschen/TransUNet)
The ISIC-2017 and CVC-ClinicDB datasets can be downloaded from offical website. [ISIC-2017](https://challenge.isic-archive.com/data/#2017) [CVC-ClinicDB](https://polyp.grand-challenge.org/CVCClinicDB/)
